Description

Your application looks good! Now, let's add the ability to upload images to your canvas. You can read Medium's tutorial on how to draw images on canvas in Javascript.

Objectives

In this stage, you need to add a handler to the input tag with the file type and the #file-input ID. Once a user uploads a file with the input file, you should take the uploaded file and draw it into the canvas. The canvas width and height should correspond to those of the image.
