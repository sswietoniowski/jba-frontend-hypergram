Description

With photo editing, there is no limit to what can be done with a photograph. You can make an unadorned landscape photo burst with colors of paradise. Any event can seem more vibrant and energetic. Your old black-and-white photographs can be colorized. Let's create a simple photo editor that will allow you to upload a photo and edit the main properties.

Objectives

Our app starts with the interface. At this stage, you need to come up with a look for your app and design a page with the following HTML components:

  • One canvas element. This is for your images.
  • An input tag with the file type and the #file-input ID. This input field will upload images to the app.
  • An input tag with the range type and the #brightness ID. This input field will adjust the image brightness.
  • An input tag with the range type and the #contrast ID. This input field will adjust the image contrast.
  • An input tag with the range type and #transparent ID. This input field will adjust the image transparency.
  • A button with #save-button ID. This button will be used to download result images.
