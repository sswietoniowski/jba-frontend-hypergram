Description

The last step is to save the resulting image. Let's add the ability to save the resulting image when the Save Image button is pressed. Here is a great tutorial by Instructobit on how to download an image from the canvas.

Objectives

When users press the button with the #save-button ID, activate the download of the current canvas image. The width and the height of the image should correspond to those of the original image. Name the resulting file result.png.
